Kebijakan Privasi

Nur al-Huda Digital Project adalah afiliasi ICC-Jakarta.

Dengan mendaftarkan diri di aplikasi web ini, Anda setuju dengan kebijakan privasi ini.

Informasi yang kami kumpulkan

  • Nama (Tidak harus nama asli)
  • Alamat Email
  • Password

Bagaimana kami menggunakan informasi-informasi tersebut

  • Informasi mendasar tersebut diperlukan untuk sistem manajemen akun aplikasi web ini berjalan, untuk kenyamanan Anda sendiri dalam menggunakan aplikasi web ini.
  • Kami menyimpan data pribadi Anda berupa nama (tidak harus nama asli), alamat email, password ketika Anda mendaftar, serta data pembelian buku elektronik Anda ketika Anda melakukan pembelian.
  • Kami menyimpan data-data tersebut untuk keperluan Anda sendiri, seperti agar bisa login di lain hari, di perangkat lain serta kenyamanan Anda dalam menggunakan aplikasi web ini.
  • Kami berkomitmen untuk menjaga data-data Anda dan tidak membagikannya ke pihak ketiga manapun.
